🎬 DEMO 流程（Demo 帳號：林珍珠，1,250 點）
1
手機瀏覽器打開 bubblecha.propskynet.com
首頁看到「林珍珠」會員卡，顯示 1,250 點
2
點橘色大按鈕「點數兌換扭蛋」
或選「使用活動資格」（不扣點數）
3
確認扣除 50 點 → 點「確認，產生條碼」
4
畫面顯示 QR Code（30 秒有效）
將手機螢幕對準機台掃描頭（QR Code 朝向掃描頭）
5
機台掃到 → 3–5 秒後自動出球 ✅
畫面自動跳到「兌換成功」
ℹ️Demo 模式可隨時從 Admin 後台補回點數與資格，不用等出球完成。

❓ 常見問題
需要下載 App 嗎？
不需要！用手機瀏覽器直接打開就可以用，不佔空間。
一次可以轉幾顆？
目前設定一次兌換一顆，要再玩可以重新兌換。
條碼過期怎麼辦？
條碼只有 30 秒，過期重新兌換就好。點「補回點數與資格」可立即補回，不影響條碼紀錄。
機器怎麼連線的？
透過 WiFi 連接雲端系統，用 MQTT 協議即時通訊，從驗證到出球只要 3 秒。
🔧 故障排除
問題處理方式
App 不顯示會員資料確認 WiFi 網路正常
條碼過期重新兌換，Demo 自動補回
扭蛋機沒出球檢查機器電源和 WiFi
點數用完Admin 後台 → Demo 控制 → 補回點數與資格
要從頭 DemoAdmin 後台 → Demo 控制 → 全場重置
